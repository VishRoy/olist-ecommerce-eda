Exploratory Data Analysis on Brazilian E-Commerce (Olist Dataset)

# 🛒 Olist E-Commerce EDA

## Overview
Exploratory data analysis on 100k orders from Brazil's 
largest e-commerce platform (2016–2018).

## Key Findings
- Olist revenue grew 3x through 2017 with a Black Friday spike
- All 27 states receive orders before estimated delivery date
- Customers in remote states wait 3.5x longer than São Paulo

## Dataset
Brazilian E-Commerce Public Dataset by Olist
https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce

## Tools Used
Python, Pandas, Matplotlib, Seaborn, Jupyter Notebook

- notebooks/ → Main EDA notebook
- images/    → Key visualizations
